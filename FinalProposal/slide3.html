DESIGN STRATEGY & FUNCTIONS

Design Strategy: On the home page, the user will be able to hover over any member's name and a dropdown will appear that lets them choose to view their profile or a link to their solo performance. The bottom right will have a drawer that introduces that group, and the button at the bottom will whirl open a lightbox with the music video.

Functions: I will be using semantic HTML and reponsive CSS for the overall structure of the website. And for the interactions and animations I will be implementing various Javascript interactions and JQuery Effects. For example, I will most likely be using Javascipt for the collapsible menu on each member's profile page and will be using JQuery for the drawer in the main page.
